# data_analyst_assistant

This is a Python-based local analytics agent that I am building using Ollama. Right now the main focus is data cleaning and SQL exploration on a local machine. The project is still being worked on and I plan to add more features as I go.

## What this project is

I wanted to build something that feels more practical and closer to real data work instead of just making a basic script. The idea is to create an agent that can help with common data analyst tasks like loading data, checking it, cleaning it up, and running SQL queries in a simpler way.

## What it can do right now

- work with data on a local machine
- help with basic data cleaning
- run SQL exploration
- use Ollama as part of the local setup

## What I plan to add

- better natural language interaction
- more automation for cleaning data
- better dataset profiling
- more features for exploring data
- possibly reporting or dashboard support later

## Tools used

- Python
- Pandas
- DuckDB
- Ollama

## Project status

This project is still in progress. I am still building it and adding more features over time.

## Why I made it

I made this project because I wanted to build something that mixes data analytics, automation, and local AI tools in a way that feels useful and hands on.
